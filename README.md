# Python Grade System

A simple Python program that accepts a student's mark (0-100) and displays the corresponding grade.

## Features

- Accepts numeric input
- Validates input range (0-100)
- Handles invalid text input
- Uses if-elif-else statements
- Uses try-except for exception handling

## Technologies

- Python 3.14.6
- Visual Studio Code
